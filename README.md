# タイ移住生活費シミュレーター

タイ移住・長期滞在を考える人が、月額生活費と生活バランスを概算するための Streamlit アプリです。

単に生活費を足し算するのではなく、

- 月いくらで、どんなタイ生活になるのか
- 車や医療保険を入れると、何が変わるのか
- 月10万円・15万円・20万円・30万円・40万円で、何が残り、何が削られるのか

を見える化することを目的にしています。

---

## 主な機能

- 生活スタイル別の月額生活費試算
- 月額予算からの日常生活費配分
- 医療保険・車・バイク維持費の概算反映
- レーダーチャートによる生活バランス可視化
- 1ページ内のステップ式動線（STEP 1 共通条件 → STEP 2 試算方法を選ぶ → STEP 3 結果を見る）
- STEP 2 で選んだ試算方法の結果だけを表示
- 予算水準・医療保険なし・車あり・最小生活に応じた注意コメントを自動表示

## セットアップと起動方法

このフォルダー内で、以下を順に実行してください。

```bash
# 1. 仮想環境の作成（初回のみ）
python3 -m venv .venv

# 2. 仮想環境の有効化
source .venv/bin/activate

# 3. ライブラリのインストール（初回のみ）
pip install -r requirements.txt

# 4. アプリの起動
streamlit run app.py
```

起動するとブラウザが自動で開きます。開かない場合は、ターミナルに表示される
`Local URL`（例：`http://localhost:8501`）をブラウザで開いてください。

終了するときは、ターミナルで `Ctrl + C` を押します。

## 使い方

1. **STEP 1** で共通条件（年齢、為替、家賃、医療保険、車・バイク、生活スタイルなど）を設定します。
2. **STEP 2** で試算方法（生活から試算する／予算から試算する）を選びます。
3. **STEP 3** で、選んだ試算方法の結果を確認します。
   - 「生活から試算する」：希望する生活スタイルでの月額生活費と内訳。
   - 「予算から試算する」：月額予算スライダーを入力し、固定費を差し引いた後の日常生活費の配分・日常生活費充足率・予備費。

STEP 1 の共通条件は両方の試算方法で共通利用され、試算方法を切り替えても入力し直す必要はありません。

### 生活スタイルについて

生活スタイルは、家賃・管理費、医療保険、医療費自己負担、光熱・通信費、ビザ費用、
車・バイク維持費を **除いた** 日常生活費です。

- **最小生活**：家賃以外をかなり抑え、外食・交際・趣味は最小限
- **控えめ生活**：基礎生活を保ちつつ、外食や趣味は少なめ
- **標準生活**：食事・外食・交際・趣味をある程度残す自然な生活
- **余裕生活**：外食・交際・趣味・レジャーにも余裕を持たせる生活

## 注意

このシミュレーションは、タイ生活費の目安をつかむための概算です。実際の支出、為替、保険料、物価、制度変更、健康状態、家族構成などにより結果は変わります。医療保険料は Pacific Cross Premier Plus を参考にした概算であり、実際の見積額ではありません。
